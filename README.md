This Android application provides the user the ability to complete the following unit conversions 

**Miles(mi) to Kilometers(km)**

**Pounds(lb) to Kilograms(kg)**

**Celisus(C) to Fahrenheit(F)**


### Usage

To use this application simply select a tab for the desired converter(e.g. distance)

Next, input a number to convert

Finally, press the desired conversion button and the conversion will be displayed at the top
